Proyek Skripsi/Kerja Praktik

Sistem Early Warning Bencana Banjir Berbasis IoT dan Notifikasi Otomatis Via Whatsapp API

Sebuah sistem peringatan dini banjir yang memanfaatkan teknologi IoT untuk mendeteksi ketinggian air dan mengirimkan notifikasi otomatis melalui WhatsApp API.

Teknologi yang Digunakan:

  • Sensor ultrasonik untuk mendeteksi ketinggian air
  • Microcontroller (Arduino/ESP8266/ESP32)
  • Platform IoT untuk pemrosesan data
  • WhatsApp Business API untuk notifikasi
  • Web dashboard untuk monitoring

Fitur Utama:

  • Monitoring ketinggian air secara real-time
  • Notifikasi otomatis ketika mencapai level bahaya
  • Web dashboard untuk visualisasi data
  • Histori data ketinggian air

Open Source Philosophy

Pandangan tentang Open Source

Saya menganut filosofi open source yang menekankan pada kolaborasi, transparansi, dan akses terbuka terhadap pengetahuan dan teknologi.

Dalam pengembangan software, saya memprioritaskan penggunaan tools open source seperti GitHub untuk version control, Figma untuk desain UI/UX, dan berbagai library open source untuk mempercepat proses development.

Penerapan dalam Projek

Dalam proyek skripsi saya, saya memanfaatkan berbagai platform open source:

  • Arduino IDE untuk pemrograman microcontroller
  • Platform IoT berbasis open source untuk pemrosesan data
  • Library open source untuk integrasi dengan WhatsApp API
  • Tools kolaborasi seperti GitHub untuk version control
